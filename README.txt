
= Goals =
 - provide infrastructure and helper classes to simplify development of Trac 
   plugins and other software which uses Trac as a platform
 - support all recent versions of Trac, provide monkey patches for older Trac 
   versions to fix nasty bugs
 - support all versions of Python where Trac runs
    - Trac 0.11: Python 2.3-2.7
    - Trac 0.12+: Python 2.4-2.7
